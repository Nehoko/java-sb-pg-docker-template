## Template for Java Spring Boot Application with PostgreSQL as database and Docker as containerization system

### Requirements
- docker
- docker-compose
- java 17

### Build
#### MacOS or Linux
`./build.sh`

#### Windows
`.\build.cmd`

### Run
#### MacOS or Linux
`./start.sh`

#### Windows
`.\start.cmd`

### Test
#### MacOS or Linux
`./test.sh`

#### Windows
`.\test.cmd`
